
# Colorado Species Occurrence Explorer (Shiny)

This is my final project Shiny app for the course.  
The app allows users to interactively filter species occurrence records
in Colorado by:

-   Species
-   Observation type
-   Elevation range
-   Year range
-   Month range

The filtered points are displayed on an interactive Leaflet map along
with the Rocky Mountain National Park (ROMO) boundary polygon.

## How to run

1.  Download/clone this repository
2.  Make sure `app.R` and `shinyDemoData.RData` are in the same folder
3.  In RStudio, open `app.R` and click **Run App**

## Packages

shiny, leaflet, sf, dplyr
